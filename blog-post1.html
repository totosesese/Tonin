Découverte HTML & CSS

J'ai découvert à quoi ressemble le code des sites internet : le HTML et le CSS ! Ce sont les langages de programmation qui permettent qu'un site s'affiche, comme celui que vous voyez en ce moment.

  • Le code HTML contient des balises qui permettent de faire des blocs de texte, des liens, des images et plein d'autres choses !
  • Le code CSS permet de mettre des polices d'écriture spéciales, des couleurs, des espacements : bref, tout ce qui est design !
